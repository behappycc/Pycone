在Mac環境下安裝Python與Sublime Text3

這篇文章會教大家如何在你的Mac上面安裝執行Python的環境與寫程式的環境，分別會安裝Python3.5與Sublime Text3

今天要介紹的是Python開發環境的安裝，不過既然提到了Python的安裝，那麼就會牽扯到一個討論已久的問題，那就是Python2跟Python3到底應該怎麼選呢，我在這邊幫大家分析一下，首先，Python2.7是Python2的最後一個版本，且預計僅支援到2020年，也就是Python2已經不會推出新的版本了，而且甚至到了2020年之後就不會在維護了，有鑑於此，很多原本只支援Python2的程式庫或框架都在不斷的往Python3遷移，使得Python3也有越來越多的第三方支援了，而且Python3的更新也十分穩定，以平均每隔一年左右的時間推出一個3.x版本，像是現在已經推出到了3.5版，因此，我們接下來的將會以Python3.5來做示範

好的，那麼我們接下來進行Python的安裝，在Chrome上面搜尋Python之後，可以查到Python Releases for Mac OS X，可以看到他有Python3.5.2版本跟Python2.7.12版本的選擇，那麼Python2跟Python3的比較剛剛已經提過了，所以我們直接開始下載3.5版本

下載完成之後直接點擊，因為是Mac的關係，在安裝上都是十分簡單的，不停的繼續，就可以了，看到這個已成功安裝的字樣那就是Python安裝成功了

利用REPL來寫第一個Python的程式吧，首先打開Mac的terminal

並在terminal輸入Python3, 這模式也被稱為Python shell，是一個Python的互動式介面，每輸入一行簡單的Python程式碼，它就會幫你進行運算並顯示結果給你

我們先從簡單的1+1開始吧，顯示結果為2，而如果你在REPL中犯錯了，REPL會有提示的訊息，練習看這些錯誤訊息是訓練寫程式的能力一個很重要的一環，接下來我們來介紹help()函式，輸入help()之後可以進入到說明模式，在這裡可以查詢一些文件跟資料，比如說你想知道python有哪些keywords，你可以輸入keywords，它就會把python裡面有定義的關鍵字都列出來，又比如你想知道剛剛使用的print()函式的定義，這裡一樣可以看到，而離開這個模式的方法是輸入quit，來返回到REPL的模式，而離開REPL模式的方法則是輸入quit()，多了一對括號要特別注意，以上就是REPL模式的一些簡單操作示範

你們會不會覺得REPL模式不好用呢，每一行都要運行一次，那如果想要一次打好幾行再執行的話，或者是比較複雜的結構該怎麼辦，這時候就是一款好的編輯器登場的時候了，在這裡為大家推薦一款編輯器，Sublime text3，直接google sublime就可以看到了，而安裝的部分我就不細講了，直接一直下一步就好，Sublime text3可以幫你處理Python要求的縮排問題，也可以幫你檢查你的語法是否有錯，它也可以幫你自動補完剩下的字，或者是幫助你知道有那些函式可以使用，另外，Sublime的背景是黑色的，長時間寫程式的話對眼睛的負擔也比較小

教大家一個好用的小技巧，就是開啟側邊欄，開啟之後我就可以把一個資料夾給拉進去，就可以很輕易地看到整個資料夾的結構，要打開其中的檔案也會十分的容易

另外，現在大家電腦的螢幕都越來越大，一次只顯示一個檔案的話有點太浪費了，在這邊教大家如何開啟Sublime裡面的雙Column，按下View裡面的Layout就可以調整Column的數量，兩個Column的話是比較適合的一個開發環境，一個用來放自己主要編輯的檔案，另外一個則可以用來放需要參考的檔案

接著我們來做一些其他的設定，按下Preferences裡面的setting，會有Default與User兩個檔案，這裡面有很多的設定可以調整，像是字體的大小、要不要顯示行數、一個tab是幾個空白等等，如果我們想要調整字體大小的話，可以將default檔案複製font_size，貼到User檔案裡，並且將其值改為你習慣的字體大小，來讓自己寫程式的過程更為舒適，另外，我們還需要將translate_tabs_to_spaces設定為true，這也是為了更好的符合Python要求的縮排規則，修改好之後要記得存檔，你可以看到檔按上面的小圓點的話，就表示這個檔案有變更但是還沒有存檔，按下Ctrl+s存檔之後小圓點就會不見了

Sublime在安裝完之後，一定是要安裝超級好用的Package Control套件，Sublime裡面有一套它的套件管理系統，點選Tools -> Install Package Control，安裝完成之後你可以在Preferences裡面看到Package Control的話那就表示你安裝成功了，之後如果你想要安裝什麼套件的話都可以透過其中的Install Package來安裝

這篇文章教大家如何在你的Mac上面安裝執行Python的環境與寫程式的環境，分別會安裝Python3.5與Sublime Text3，當環境都安裝好後，就可以開始來寫Python囉!

“Happy coding.”
