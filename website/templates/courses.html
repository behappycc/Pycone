初心者 - Python入門

本課程適合的對象為1.   想學程式卻不知道該從什麼語言入門的人2. 已經有學過如C, c++, java但是想要學習Python的人3. 對網站或是資料分析有興趣,想要自己動手做做看，但是還在尋找適合的語言來實作的人。我們這門課適合Python的初學者，學生們並不需要具備程式的開發經驗或是Python的使用經驗這門課會從最一開始的環境設定，到簡單的語法與資料結構的教學，以及物件導向程式設計的概念, 並在最後搭配一個實務專題來讓學生們能夠從無到有 學會Python的完整開發流程。 完整課程資訊

Python 網頁爬蟲入門實戰

你會學到如何撰寫 Python 與網站或 API 溝通，解構網頁，擷取資訊，儲存資料，有系統寫出爬蟲程式，並完成課程範例 (PTT 網頁爬蟲、Yahoo 奇摩電影爬蟲、FB Graph API 爬蟲等等)，進而寫出屬於你自己需求的爬蟲程式。網路上有許多網頁爬蟲的教學文章與影片，它們都是很好的自學素材，但是這些素材多為片段，常常直接切入主題，沒有循序漸進的概念建構，本課程是針對網路爬蟲的完整教學，由淺入深，搭配實戰範例與專題練習，力求符合學生的需要。講者有各種網頁及手機 App 的爬蟲撰寫經驗，也會在課程中分享個人遇過的困難及處理經驗，解答學生的問題。 完整課程資訊
